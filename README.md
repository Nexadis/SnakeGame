# SnakeGame

Игра змейка с возможностью настройки размера поля и скорости игры.

## Управление

`ArrowUp`     `w` - поворот *вверх*

`ArrowLeft`   `a` - поворот *налево*

`ArrowBottom` `s` - поворот *вниз*

`ArrowRight`  `d` - поворот *направо*
